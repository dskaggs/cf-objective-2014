CFMongoDB

  • Could just use Java driver but why reinvent the wheel
  • Easy to use custom DSL or API built in
  • Multiple people already have done a lot of work to make it easy to communicate with MongoDB through the Java driver
  • Allows nearly seamless translating of ColdFusion structs and arrays to MongoDB and vice versa
  • Works with ColdFusion 9.01+ and Railo 3.2+
